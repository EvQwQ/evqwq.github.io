你好！这是我的第一篇博客文章。

这个博客使用 Hugo 构建，并托管在 GitHub Pages 上，完全免费！

特点

  • ⚡ 超快 - Hugo 生成速度极快，几秒钟就能构建整个网站
  • 🎨 美观 - 简洁现代的设计，支持深色模式
  • 📱 响应式 - 完美适配手机、平板、电脑
  • 🏷️ 标签系统 - 文章可以添加标签，方便分类浏览
  • 📂 分类归档 - 文章可按分类归类，便于查找
  • 🆓 免费 - GitHub Pages 免费托管

技术栈

  • Hugo - 静态网站生成器
  • GitHub Pages - 免费托管
  • GitHub Actions - 自动部署
