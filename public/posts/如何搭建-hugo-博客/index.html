本文将详细介绍如何使用 Hugo 搭建一个漂亮的博客，并部署到 GitHub Pages。

什么是 Hugo？

Hugo 是一个用 Go 语言编写的静态网站生成器，以极速构建著称。它使用 Markdown 格式编写内容，通过主题模板生成 HTML 页面。

安装 Hugo

Windows

1

winget install Hugo.Hugo

macOS

1

brew install hugo

创建新站点

1
2
3

hugo new site myblog
cd myblog
git init

添加主题

1

git submodule add https://github.com/CaiJimmy/hugo-theme-stack.git themes/stack

然后在 hugo.toml 中设置 theme = "stack"。

创建文章

1

hugo new content posts/hello.md

本地预览

1

hugo server -D

访问 http://localhost:1313 预览。

部署到 GitHub Pages

  1. 创建仓库 username.github.io
  2. 启用 GitHub Actions
  3. 推送代码

更多细节请参考官方文档。
