如何搭建 Hugo 博客

2026年3月18日
hugo github-pages

本文将详细介绍如何使用 Hugo 搭建一个漂亮的博客，并部署到 GitHub Pages。

什么是 Hugo？

Hugo 是一个用 Go 语言编写的静态网站生成器，以极速构建著称。它使用 Markdown 格式编写内容，通过主题模板生成 HTML 页面。

安装 Hugo

Windows

winget install Hugo.Hugo

macOS

brew install hugo

创建新站点

hugo new site myblog
cd myblog
git init

添加主题

git submodule add https://github.com/theNewDynamic/gohugo-theme-ananke.git themes/ananke
echo "theme = 'ananke'" >> hugo.toml

创建文章

hugo new content posts/hello.md

本地预览

hugo server -D

访问 http://localhost:1313 预览。

部署到 GitHub Pages

  1. 创建仓库 username.github.io
  2. 启用 GitHub Actions
  3. 推送代码

更多细节请参考官方文档。

欢迎来到我的博客

2026年3月18日
hugo github-pages

你好！这是我的第一篇博客文章。

这个博客使用 Hugo 构建，并托管在 GitHub Pages 上，完全免费！

特点

  • ⚡ 超快 - Hugo 生成速度极快，几秒钟就能构建整个网站
  • 🎨 美观 - 简洁现代的设计，支持深色模式
  • 📱 响应式 - 完美适配手机、平板、电脑
  • 🏷️ 标签系统 - 文章可以添加标签，方便分类浏览
  • 📂 分类归档 - 文章可按分类归类，便于查找
  • 🆓 免费 - GitHub Pages 免费托管

技术栈

  • Hugo - 静态网站生成器
  • GitHub Pages - 免费托管
  • GitHub Actions - 自动部署

开始写作

创建新文章非常简单：

hugo new content posts/文章标题.md

然后编辑文章，添加分类和标签：

---
title: "文章标题"
date: 2026-03-18T12:00:00+08:00
draft: false
categories: ["技术"]  # 分类
tags: ["hugo", "教程"]  # 标签
---
