欢迎来到我的博客

2026年3月18日
hugo 博客 github-pages

你好！这是我的第一篇博客文章。

这个博客使用 Hugo 构建，并托管在 GitHub Pages 上，完全免费！

特点

  • ⚡ 超快 - Hugo 生成速度极快，几秒钟就能构建整个网站
  • 🎨 美观 - 简洁现代的设计，支持深色模式
  • 📱 响应式 - 完美适配手机、平板、电脑
  • 🏷️ 标签系统 - 文章可以添加标签，方便分类浏览
  • 🆓 免费 - GitHub Pages 免费托管

技术栈

  • Hugo - 静态网站生成器
  • GitHub Pages - 免费托管
  • GitHub Actions - 自动部署

开始写作

创建新文章非常简单：

hugo new content posts/文章标题.md

然后编辑文章，添加标签：

---
title: "文章标题"
date: 2026-03-18T12:00:00+08:00
draft: false
tags: ["标签1", "标签2"]
---

感谢你的访问！欢迎常来看看 👋
