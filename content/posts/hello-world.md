你好！这是我的第一篇博客文章。

这个博客使用 [Hugo](https://gohugo.io/) 构建，并托管在 GitHub Pages 上，完全免费！

## 特点

- ⚡ **超快** - Hugo 生成速度极快，几秒钟就能构建整个网站
- 🎨 **美观** - 简洁现代的设计，支持深色模式
- 📱 **响应式** - 完美适配手机、平板、电脑
- 🏷️ **标签系统** - 文章可以添加标签，方便分类浏览
- 📂 **分类归档** - 文章可按分类归类，便于查找
- 🆓 **免费** - GitHub Pages 免费托管

## 技术栈

- [Hugo](https://gohugo.io/) - 静态网站生成器
- [GitHub Pages](https://pages.github.com/) - 免费托管
- [GitHub Actions](https://github.com/features/actions) - 自动部署

## 开始写作

创建新文章非常简单：

```bash
hugo new content posts/文章标题.md
```

然后编辑文章，添加分类和标签：

```yaml
---
title: "文章标题"
date: 2026-03-18T12:00:00+08:00
draft: false
categories: ["技术"]  # 分类
tags: ["hugo", "教程"]  # 标签
---
```
