你好！这是我的第一篇博客文章。

这个博客使用 [Hugo](https://gohugo.io/) 构建，并托管在 GitHub Pages 上，完全免费！

## 特点

- ⚡ 超快的生成速度
- 🎨 简洁美观的设计
- 📱 响应式布局，支持移动端
- 🆓 免费托管在 GitHub Pages

## 开始写作

你可以使用以下命令创建新文章：

```bash
hugo new content posts/新文章标题.md
```

然后编辑 `content/posts/新文章标题.md` 文件即可。

感谢你的访问！
