## 关于这个博客

这个博客使用 [Hugo](https://gohugo.io/) 构建，并托管在 GitHub Pages 上，**完全免费**！

### 特点

- ⚡ **极速** - Hugo 生成速度超快，每秒可生成数千页面
- 🎨 **美观** - 现代化的设计，支持深色模式
- 📱 **响应式** - 完美适配手机、平板、电脑
- 🆓 **免费** - GitHub Pages 免费托管
- 📝 **Markdown** - 使用 Markdown 写作，简单高效

## 代码示例

支持语法高亮的代码块：

```python
# Python 示例
def hello_world():
    print("Hello, World!")
    return "欢迎访问我的博客"

result = hello_world()
```

```javascript
// JavaScript 示例
const greeting = () => {
    console.log("Hello, World!");
    return "欢迎访问我的博客";
};

greeting();
```
