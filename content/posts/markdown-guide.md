Markdown 是一种轻量级标记语言，让你专注于写作而不是排版。

## 基础语法

### 标题
```markdown
# 一级标题
## 二级标题
### 三级标题
```

### 文本样式
- **粗体**：`**文字**`
- *斜体*：`*文字*`
- ~~删除线~~：`~~文字~~`

### 列表

无序列表：
- 项目 1
- 项目 2
- 项目 3

有序列表：
1. 第一步
2. 第二步
3. 第三步

### 代码

行内代码：`code`

代码块：
```python
print("Hello, World!")
```
